# jajko-vs-wentylator
Bot discordowy, który wybiera czy obrazek bardziej wygląda jak jajko, bądź wentylator. Ma też kilka innych funkcji.
Aby użyć:

1. Zainstalować bibliotekę discord i keras
2. Stworzyć bota używając discord developers
4. Wpisać własny token discorda
5. Uruchomić przy pomocy konsoli (python main.py)

Możesz teraz używać swojego bota do woli!

# jajko-vs-wentylator
Bot discordowy, który wybiera czy obrazek bardziej wygląda jak jajko, bądź wentylator. Ma też kilka innych funkcji.
Aby użyć:
1. Skopiować pliki do oddzielnego folderu
2. Wpisać własny token discorda
3. Uruchomić przy pomocy konsoli (python model.py)

Możesz teraz używać swojego bota do woli!
